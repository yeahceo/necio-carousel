## 0. 핵심 약속 (모든 결정의 기준)

> ## "들어올 때 선언한 목표를, 2달 안에 진짜 이루고 나간다."

- *물류센터 자동화를 목표로 오셔서, 진짜 AI로 자동화하고 나가셨습니다.*
- *전자책 판매를 목표로 오셔서, 2달 만에 600만원을 벌고 나가셨습니다.*
- **"당신은 무엇을 이루고 나가고 싶으세요?"**

넥씨오는 0을 1로 만드는 곳이 아니라, **목표를 가진 사람을 진짜 결과까지 끌고 가는 실행 엔진**이다.
이 한 문장에서 모집·면접·가격·활동·이탈관리가 전부 파생된다.

## 2. 선발 — 두 트랙으로 받는다

아이템 유무로 거르지 않는다. **2달 안에 측정 가능한 결과를 만들 사람**인지로 거른다.

| | A트랙 — 아이템 보유 | B트랙 — 성실하나 아이템 없음 |
|---|---|---|
| 2달 목표 | 가진 아이템으로 직접 성과 | 강점 기반 인스타 주제 확정 후 계정 시작 |
| 결과 예시 | 전자책 첫 매출, 창고 자동화 1건 | 주제 확정 + 20개 발행 + 팔로워 300 + 첫 문의/판매 1건 |
| 근거 | 기존 성공 사례 | 드로우앤드류·윤소정式 퍼스널 브랜드 모델 |

- **공통 원칙:** 두 트랙 모두 "2달 뒤 숫자·결과물로 측정 가능한 목표"로 끝낸다.
- **B트랙 주의:** "계정 시작"은 활동이지 결과가 아니다. 반드시 숫자(발행 수·팔로워·첫 반응)로 못 박는다.
- **유일한 탈락 대상:** 아이템도 없고, 주제 잡고 꾸준히 할 성실함도 없는 사람 = "물 흐리는 사람"

### 면접 흐름
교육 안내 링크 공유 → 사전 질문 → 온라인 미팅(왜 하고 싶은지) → 이 자리에서 가격 안내 → 상호 결정
(별로면 안 받는다. 이 과정에서 떨어져 나가는 사람도 자연스럽다. 끝까지 열정적으로 할 사람만 받는다.)

### 핵심 면접 질문
1. **"2달 뒤, 뭐가 숫자나 결과물로 달라져 있으면 성공인가요?"** (측정 가능성)
2. 그 목표에 **절박함**이 있는가 (2주차 이탈은 결국 이유가 약해서)
3. 최근 뭔가를 **끝까지 해본 경험**이 있는가 (의지 표명보다 정확한 신호)

### 합격자 기준 (정량)
- 모임 8개(온라인2·오프라인2·간단목표선언4) 중 6회 이상 출석할 사람
- 매일 챌린지 40회 중 35회 이상 인증할 사람
- 열정의 물을 흐리지 않을 사람

## 3. 활동 구조 (전부 "목표 달성"으로 꿰어짐)

| 활동 | 빈도 | 의미 |
|---|---|---|
| 매일 인증 (노션) | 매일 / 목표 40회 중 35회 | "오늘 목표를 향해 한 발 갔는가" |
| 주간 공개 다짐 미팅 | 매주 (온라인 → 노션 업로드) | 공개 선언으로 책임감 |
| 1:1 콜 (온라인 미팅) | 월 4회 / 45분 고정 | "목표가 막혔을 때 뚫어주는 자리" |
| 온라인 모임 | 2회 | |
| 오프라인 모임 | 2회 | 비전문·사명문 / 원씽 캐스케이딩 |
| 간단 목표선언 모임 | 4회 | |
| 사업 스타터 챌린지(사스챌) | 운영 | 초반 시작 동력 (B트랙 주제잡기에 최적) |

- **콜은 "원하면"이 아니라 월 4회 45분으로 고정.** 안 쓰는 시간은 운영자가 회수.
- **단톡방은 A·B트랙 하나로 통합.** 4명을 쪼개면 단톡이 죽는다. 트랙 차이는 단톡이 아니라 **1:1 콜과 사스챌에서 흡수.** 분야가 다른 게 서로 자극이 된다.

## 4. 이탈 방지 구조 (선발만으로 안 됨)

> 2주차 이탈이 고질병. 이전엔 보증금이 없어서 더 그랬다.
> 이탈을 막는 건 선발보다 **구조**다.

**① 보증금 환급 조건 명문화** — 면접 때 미리 고지
- "출석 6회 + 인증 35회 + [후기 스토리 + 졸업 후기] 달성 시 5만원 전액 환급"
- 미달 시 예사장이 DM ❌ → 보증금에서 자동 차감 (추궁할 일 자체가 사라짐)

**② "왜 못했나" 대신 "내일 뭐 할까"**
- 이유 추궁 ❌ (변명·죄책감 → 단톡 회피 → 역효과)
- "내일 다시 시작할 때 한 줄만: 내일은 뭘 할 건지" → 재시작을 쉽게 (원씽 철학과도 일치)

**③ 단톡 공개 자리로 사회적 압박 분산**
- "오늘 스토리 인증의 날!" → 올린 사람이 자랑하는 자리 → 안 한 사람이 알아서 눈치
- 사람은 돈보다 사람 눈치에 더 움직인다.

## 5. 후기 / 책 이벤트 (역할 재정의)

- **유입용 책 이벤트는 제외.** 책 받으러 오는 결의 사람은 면접에서 거를 대상.
- 출판사 협업은 **관계 유지 + 기수 중 깜짝 보너스**로만 (부담 없을 때).
- **후기는 책이 아니라 "클럽 경험"으로 분리.** 졸업 시 "2달 회고 한 장"이 다음 기수 모집의 진짜 무기.
- 스토리 안 올리는 진짜 이유 = "뭘 올릴지 몰라서" → **태그(@yeah_ceo @next_ceo_club) 박힌 템플릿** 미리 제공.

## 6. 운영자 지속가능성 (가장 중요)

> 회원 이탈보다 **운영자 번아웃이 클럽을 죽이는 진짜 원인.**
> 예사장은 8-5 직장 병행 → 가진 시간은 저녁·주말뿐.

- 콜 월 4회 고정 → 시간 예측 가능
- 반복 작업(인증 체크·회고 정리)은 **노션 연동으로 클로드에게 위임** → 예사장 시간은 콜과 교육에만
- 정원 4명 유지로 손길 관리 가능

## 7. 확장성 전략 (8-5 직장인 버전)

> 직장인에게 "확장"은 사람 수가 아니다. 1:1 콜은 사람 수에 정비례해 저녁 시간을 먹는다.
> **확장 = 사람 수 ❌ / 사람당 가치 ⭕ + 시스템 ⭕ + 졸업생 레버리지 ⭕ + 콘텐츠 자산 ⭕**

병목 = 1:1 콜 (가장 값지지만 가장 안 늘어남)

**1단계 — 지금 3기 (수작업으로 증명)**
4명 고밀착. 지금은 확장 안 돼도 됨. 목표: 킬러 후기 확보 + 반복 가능한 플레이북 정리.

**2단계 — 시스템화 (운영자 시간 빼기)**
- 비전문·캐스케이딩·사스챌 = 녹화/템플릿화
- 인증 체크·회고 정리 = 노션+클로드 자동화
- 같은 4~6명을 훨씬 적은 시간으로 운영

**3단계 — 졸업생을 코치로 (레버리지)**
각 기수 열심히 한 졸업생이 다음 기수 멘토/리더로. B트랙 초반 코칭 분담.

**4단계 — 그룹 콜 전환 (선택)**
1:1 → 그룹 코칭. "기본 그룹 콜 + 가끔 1:1" 하이브리드.

**병행 자산:** 고객 인터뷰 23건 + 04 new brain 지식 = 시간 안 먹고 쌓이는 확장 자산 → 책/강의/콘텐츠로 전환.

→ 결론: 22만원 4명은 지금은 일부러 안 키운다. 1단계(증명)로 플레이북 확보 후 2~3단계에서 시간당 가치를 올린다.
